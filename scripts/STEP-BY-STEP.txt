┌─────────────────────────────────────────────────────────────┐

┌─────────────────────────────────────────────────────────────┐
│ STEP 2: View Page Source                                   │
└─────────────────────────────────────────────────────────────┘

Windows: Press Ctrl + U
Mac:     Press Cmd + Option + U

This opens a new tab with the HTML source code


┌─────────────────────────────────────────────────────────────┐
│ STEP 3: Copy EVERYTHING                                    │
└─────────────────────────────────────────────────────────────┘

Windows: Press Ctrl + A, then Ctrl + C
Mac:     Press Cmd + A, then Cmd + C

⚠️  IMPORTANT: Make sure you copied the ENTIRE page!
   - It should start with <!DOCTYPE html>
   - It should end with </html>
   - It will be VERY long (100,000+ characters)


┌─────────────────────────────────────────────────────────────┐
│ STEP 4: Paste into the Script                              │
└─────────────────────────────────────────────────────────────┘

1. Open this file in your code editor:
   FPUSA-NEXTJS-Template/scripts/paste-html-here.js

2. Find this line:
   const HTML_CONTENT = `
   PASTE THE FULL HTML HERE
   `;

3. Replace "PASTE THE FULL HTML HERE" with your copied HTML

4. Update the ALBUM_INFO:
   const ALBUM_INFO = {
     id: 'champions-gate-luxury',     // ← Change this
     title: 'Champions Gate Luxury',   // ← Change this
     location: 'Champions Gate, FL',   // ← Change this
     category: 'highlight'             // ← 'highlight' or 'standard'
   };
